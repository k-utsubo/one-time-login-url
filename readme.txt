Use WP-CLI to generate a one-time login URL for any user

== Description ==

    wp user one-time-login-url user [--count=NUM] [--delay-delete=MINUTE] [--from-date="YYYY-MM-DD HH:MM:SS"] [--to-date="YYYY-MM-DD HH:MM:SS"]
